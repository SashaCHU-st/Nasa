## Nasa program Usage
1. Clone repository
2. Move to directory
```bash
cd Nasa
```
### 3.For backend
3.1 Move to directory
```bash
cd backend
```
3.2 intall dependendecies
```bash
npm install
```
3.3 Run backend
```bash
npm start
```

### 4. For frontend
4.1 Move to directory
```bash
cd frontend
```
4.2 intall dependendecies
```bash
npm install
```
4.3 Run frontend
```bash
npm start
```
